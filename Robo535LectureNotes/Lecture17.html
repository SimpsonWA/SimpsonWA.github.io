• Discriminant Function Approach: Find a function \(f(x)\) which maps each input \(\vec{x}\) directly onto a class label
  • Simplest representation of a linear discriminant function is obtained by taking the linear function of the input vector: $$ y(\vec{x}) = \vec{w}^{T}\vec{x} + w_{0} $$
  • Then the classification rule is given by:
    • An input vector \(x\) is assigned to class \(C_{1}\) if \(y(\vec{x}) \geq 0\)
    • An input vector \(x\) is assigned to class \(C_{2}\) if \(y(\vec{x}) < 0\)
  • Given a linear discriminant function based classification model: $$ y(\vec{x}) = f(\vec{w}^{T}\vec{x}+w_{0}) $$

  • We can learn the parameters of the model by following 3 approaches:

    • Least square
    • Perceptron algorithm,
    • fisher's linear discriminant

      Least Squares for classification

  • Each class \(C_{k}\) is described by its own linear model so that : $$ y_{k}(\vec{x}) = \vec{w}_{k}^{T}\vec{x}+w_{k0} $$

  • where \(k = 1,\dots,K\)
  • We can group these together using vector notation so that: $$ y(\vec{x}) = \hat{W}^{T}\hat{x} $$
  • Where
    • \(\hat{W}\): is a matrix whose \(k\)-th column comprises the \(D+1\)-dimensional vector \(\hat{w}_{k}= (w_{k0},w_{k}^T)^{T}\)
    • and \(\hat{x}\) is the corresponding augmented input vector \((1,x^{T})^{T}\) with a dummy input \(x_{0}= 1\)
