• Uncertainty is a native property of real world data
    • Uncertainty arises from noise on measurements
    • It also arises through the finite size of data sets (incomplete observations)
  • Probability theory is the foundation of ML
  • Probability theory provides a consistent framework for the quantification and manipulation of uncertainty and forms one of the central foundations for pattern recognition and ML
  • Combined with decision theory, it allows us to make optimal predictions

Definition of Probability

Probability: - Probability or likely hood is a measure of how likely something will happen or is true - Probability values range from 0 (wont happen) to 1 (will happen) - Formal: Probability p is a positive real number describing the likelihood of an event satisfying: - for any event: \(P(X = e) \in [0,1]\) - For a mutually exclusive and exhaustive set of events (\(E\)), \(\sum_{X \in E} P(X) =1\)

Starting from a General Probability Problem

  • Consider a setting with 2 random variables

    • \(X\): which takes values \({x_i}\) where \(i = 1,2,...,M\)
    • \(Y\): which takes values \({y_i}\) where \(i = 1,2,...,L\) ![[Pasted image 20241026131756.png]]

  • In the illustration above \(M=5\), and \(L=3\)

  • If we consider a total number \(N\) of instances of these variables, then we denote the number of instances where \(X = x_i\) and \(Y=y_j\) by \(n_ij\) ; which is the number of points in the corresponding cell of the array
  • The number of points in column \(i\) corresponding to \(X=x_i\) is denoted \(c_i\)
  • The number of points in row \(j\) corresponding to \(Y = y_j\) is denoted \(r_j\)

Joint Probability

  • The probability that \(X\) will take the value \(x_i\) and \(Y\) will take the value \(y_j\) is written as $$ P(X = x_{i,}Y = y_j) $$
  • This is called the joint probability of \(X=x_i\) and \(Y=y_j\) $$ P(X = x_{i,}Y = y_{j)}= P(X=x_{i} \ \cap \ Y = y_{j )}= \frac{n_{ij}}{N} $$
