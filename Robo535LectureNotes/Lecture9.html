Learning Via Parameter Estimation

  • Basic Setting:
    • A set of random variables \(X = {X_{1},...,X_{D}}\)
    • A model of the distribution over the variables in \(X\) with parameters \(\theta : \hat{p}(X|\theta)\)
    • Data (a set of observations) \(\mathcal{D} = {x_{1},...,x_{N}}\)
  • Objective:
    • Find parameters \(\theta\) such that \(P(X|\theta)\) fits data \(\mathcal{D}\) the best

Bernoulli Distribution:

  • The distribution over \(x\) for the coin flipping example can be written as; $$ Bern(x|\mu) = \mu^{x}(1-\mu)^{1-x} \quad \quad x\in {0,1} $$
  • This is known as Bernoulli distribution
  • We can say that:
    • \(\mathbb{E}(x) = \mu\)
    • \(var[x] = \mu (1- \mu)\)

Maximum Likelihood Estimation (MLE): Coin Example

  • Assume we have a data set \(\mathcal{D} = {x_{1},..,x_{N}}\) of observed values of \(x\). We can construct the likelihood function, which is a function of \(\mu\), on the assumption that the observations are drawn independently from the \(p(x|\mu)\) so that : $$ p(\mathcal{D}|\mu) = \Pi_{i=1}^{N}p(x_{i}|\mu) = \Pi_{i=1}^{N} \mu^{x_{i}}(1-\mu)^{1-x_{i}} $$
  • In a frequentist setting we can estimate the value of \(\mu\) by maximizing the likelihood function: $$ ln(p(\mathcal{D}|\mu)) = \sum\limits_{i=1}^{N} p(x_{i}|\mu) = \sum\limits_{i=1}^{N} {x_{i}ln(\mu) + (1-x_{i})ln(1-\mu)} $$
  • Setting the derivative of \(ln(p(\mathcal{D}|\mu))\) with respect to \(\mu\) to 0 we obtain: $$ \hat{\mu}_{MLE} = \frac{1}{N}\sum\limits_{i=1}^{N} x_{i} = \frac{m}{N} $$
